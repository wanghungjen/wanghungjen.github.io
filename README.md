# Personal Academic Website

My personal academic website and portfolio, built using the **Academic Pages** Jekyll template and hosted with **GitHub Pages**. The site serves as a central hub for my research, projects, publications, teaching, and professional experience.

**Live Website:** https://wanghungjen.github.io

---

## Features

* Personal homepage and biography
* Research interests and publications
* Project portfolio
* Resume / CV
* Teaching and academic activities
* Blog posts and updates
* Responsive design optimized for desktop and mobile
* Automatic deployment through GitHub Pages

---

## Tech Stack

* **Jekyll**
* **Academic Pages**
* **GitHub Pages**
* **Liquid**
* **Markdown**
* HTML / CSS / SCSS
* YAML

---

## Deployment

The website is automatically built and deployed using **GitHub Pages** whenever changes are pushed to the `main` branch. GitHub Pages provides free hosting for static sites directly from a GitHub repository.
